# Belgeler (`docs/`)

Bu klasörde mimari kurallar, geçmiş analizler ve **taslak yol haritaları** bir arada durur.

| Kategori | Not |
|----------|-----|
| **`architecture/`** | Çekirdek kurallar ve yeni modül checklist’i; canlı kodla uyumlu tutulmalı. |
| **`*-audit.md`, `*-analysis.md`** | Belirli bir zaman dilimindeki inceleme; içeride **Contact**, **Organization** gibi bu repoda artık olmayan veya henüz eklenmemiş örneklere referans olabilir. |
| **`organization-module-roadmap.md`** | Organization modülü için **taslak** plan; repoda ilgili domain kodu yoktur. |

**Kaynak doğruluk:** Davranış ve API için her zaman `src/` ve `tests/` altındaki kod esas alınmalıdır.
